# SenpaiBot 2.0

Our bot for our Discord server.

## Requirements

See makefile for more details.

- python3.10 or higher
- pip
  - discord.py
  - [webpreview](https://github.com/ludbek/webpreview)

## Running

Makefile:

```
~ $ make run
```

Manually:

```
~ $ cd SenpaiBot2.0/src
~ $ python3 senpaibot.py
```
